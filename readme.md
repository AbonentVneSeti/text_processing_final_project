# Итоговая работа по текстам

## Описание
Проект 20. Перефразирование предложений (парафраз-генерация)\
Источник данных: Парсинг параллельных корпусов из PPDB (Paraphrase Database) или генерация парафразов через back-translation (парсинг текстов + перевод через API).\
Задача: Генерация парафраза – перефразирование входного предложения с сохранением смысла.\
\
Ход решения:
1. Сбор пар (оригинал, парафраз) через back-translation (ru → en → ru) или из открытых датасетов.
2. Очистка.
3. Токенизация.
4. Построение базовой модели.
5. Настройка и обучение предобученной модели.
6. Оценка: BLEU, BERTScore, semantic similarity (cosine similarity эмбеддингов), ручная проверка.
